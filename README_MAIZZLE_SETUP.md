# Maizzle Email Project (minimal)

Quick start commands for this minimal Maizzle v5 setup.

1. Install dependencies:

```powershell
npm install
```

2. Dev server (live previews):

```powershell
npm run dev
```

3. Build production-ready inlined HTML:

```powershell
npm run build
```

Output will be in `output/`. Components are loaded from `dynamic_templates_maizzle/`.
